# SPARQ
## Was ist Sparq?
Sparq ist eine Webapp um effizienter zu lernen. Sparq ist für Schüler der 5. bis 10. Jahrgangsstufe konzipiert.

## Wie viel kostet Sparq?
Sparq ist komplett kostenlos für jeden verfügbar.

## Wie wurde Sparq programmiert?
Sparq wurde von Claude Sonnet 4.6 programmiert, da die Nutzung von KI für Code schneller und effizienter ist.

## Wann kommt das nächste Update?
Es werden regelmäßig neue Updates veröffentlicht und neue Features hinzugefügt, jedoch gibt es keine festen Zeitpläne.

## Wo kann ich Features vorschlagen oder Bugs melden?
Öffne einfach ein Issue in unserem Github und wir werden deine Anfrage prüfen.
